## Overview

This is the personal portfolio of **Kumar Aryan**, designed with a focus on brutalist precision, high-performance engineering, and a sleek, dark-mode aesthetic. The interface bridges the gap between abstract algorithmic efficiency and pixel-perfect user experiences.

### Key Features
- **Custom System Preloader:** High-tech initialization sequence.
- **Micro-Animations:** Fluid, scroll-triggered animations powered by Framer Motion.
- **Custom Cursor:** Bespoke pointer interactions that elevate the UI.
- **Architectural Aesthetic:** Minimalist typography, dark UI, and strict grid alignments.

## Tech Stack

This project is built using modern web standards for optimal performance and developer experience:
- **React 19**
- **Vite** (Build Tool & Dev Server)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Run Locally

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```sh
   git clone <your-repository-url>
   ```

2. **Navigate into the directory:**
   ```sh
   cd <your-repository-directory>
   ```

3. **Install dependencies:**
   ```sh
   npm install
   ```

4. **Environment Variables:**
   Copy the example environment file and add your own variables if required.
   ```sh
   cp .env.example .env
   ```

5. **Start the development server:**
   ```sh
   npm run dev
   ```

The application will be available at `http://localhost:3000`.
